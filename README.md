# 🛍️ Giftos E-Commerce Website

A fully responsive e-commerce website for showcasing fashion products. Built with **HTML**, **CSS**, and **JavaScript**, this template is ideal for an online clothing store.

- Responsive design for all devices (Mobile, Tablet, Desktop)

- Home, Products, About, Contact pages
- Newsletter subscription section
- Hero slider and featured products
